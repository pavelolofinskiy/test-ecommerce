# Проект Каталог Товаров на PHP

Это простой проект каталога товаров с импортом и фильтрацией, реализованный на чистом PHP с использованием MySQL и Redis.

---

## Требования

- PHP >= 7.4 с поддержкой PDO и Redis
- MySQL (или MariaDB)
- Redis сервер
